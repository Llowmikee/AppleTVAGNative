# Установка AppleTV AGNative One-Click Bundle

## 1. Скопируй файл
Скопируй:
- `appletv_agnative_bundle.js`

в папку plugin JS на сервере, например:
- `/wwwroot/plugins/appletv_agnative_bundle.js`

## 2. Подключи файл
В своём исполняемом файле / entrypoint для плагинов добавь подключение:
- `/plugins/appletv_agnative_bundle.js`

## 3. Обнови клиент / перезапусти сервис
- перезапусти сервис, если это требуется на твоём сервере
- обнови клиент / перезагрузи приложение

## 4. Включи сцену
Есть два варианта:

### Вариант A
Открой:
- `Настройки → Интерфейс → Включить AppleTV AGNative`

### Вариант B
Или вручную через выбор сцены:
- `Настройки → Интерфейс → AppleTV AGNative`

## Проверка
Если всё подключено правильно:
- сцена `AppleTV AGNative` появится в списке
- mechanics карточек будут работать
- AGNative visual layer применится автоматически

## Важно
Этот пакет рассчитан на сервер, где уже есть:
- plugin entrypoint
- базовая theme/plugin инфраструктура
- Lampa-подобный runtime
